# MoviWeb
Movie Database Web App


## How to Install
Install the needed Flask dependencies.
```
pip install -r requirements.txt
```

### API Key not included in source code
For the app to work, you must add the following variables in a file called `secret.py`:
```
###### OMDb API Key | https://www.omdbapi.com/
API_KEY = '1234abcd'
```

## How to Run
```
python app.py
```
